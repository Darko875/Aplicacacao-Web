# Aplicacao-Web
Aplicacao Web para despesas
 As instruções são as seguintes:

 	1. Despesas Mensais Fixas: coloque sempre estes valores já que são obrigatórios.
 	2. Despesas Mensais Variáveis:
		a. os valores são primeiro o minimo e depois o máximo;
		b. os valores deste tópico são o média, pois é impossível conter um valor absoluto.
	3. Desfrute do aplicativo
